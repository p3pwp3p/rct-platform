# RCT Platform — 프로젝트 기획 정리

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 플랫폼 유형 | 네트워크 마케팅 (바이너리 구조) |
| 수익 재원 | MT5 EA 카피트레이딩 profit sharing |
| 목표 수익률 | 월 평균 10% |
| Profit sharing 재원 | 월 수익의 40%를 보상 재원으로 배분 |
| 톤 | 전문적이지만 친근함, 구체적 수치 기반 |

---

## 2. 기술 스택

| 레이어 | 선택 | 비고 |
|--------|------|------|
| 프레임워크 | Next.js 15 (App Router) | SSR/SSG/CSR 혼용, 파일 기반 라우팅 |
| 스타일링 | Tailwind CSS | 빠른 UI 프로토타이핑 |
| 컴포넌트 | shadcn/ui | Tailwind 기반 고품질 컴포넌트 |
| DB / Auth | Supabase | 인증, 실시간 DB, Row Level Security 내장 |
| 상태관리 | Zustand | 가볍고 보일러플레이트 없음 |
| 배포 | Vercel | Next.js 제작사, 무료 티어로 시작 |
| 트리 시각화 | React Flow | 바이너리 레그 노드/엣지 렌더링 |
| MT5 API 연동 | 없음 | — |

---

## 3. 서비스 구조

```
RCT Platform
├── 첫 화면 (Landing Page)
├── 로그인 화면
├── 회원가입 화면
└── 메인 페이지 (Dashboard)
    ├── 본인 계정 바이너리 레그 확인
    ├── 레그 화면
    └── 계정 직급 확인 페이지
```

### 3-1. 첫 화면 (Landing Page)
- 플랫폼 소개 및 핵심 가치 제안
- MT5 카피트레이딩 기반 수익 구조 요약 (월 목표 10%, 보상 재원 40%)
- CTA: 로그인 / 회원가입 버튼
- 현재 플랫폼 참여자 수, 누적 profit sharing 금액 표시

### 3-2. 로그인 화면
- 이메일 + 비밀번호 입력 (Supabase Auth)
- 비밀번호 찾기 링크
- 소셜 로그인 (선택)
- 로그인 후 → 메인 페이지로 이동

### 3-3. 회원가입 화면
- 입력 항목: 이름, 이메일, 비밀번호, 추천인 코드 (referral code)
- 약관 동의
- 가입 완료 후 → 메인 페이지로 이동

### 3-4. 메인 페이지 (Dashboard)
- **본인 정보 요약**: 직급, 현재 레그 현황, 이번 달 profit sharing 예상액
- **빠른 링크**: 레그 확인 / 직급 확인 / 수익 내역
- **공지 및 업데이트** 섹션
- **메뉴 내비게이션** → 하위 섹션으로 이동

### 3-5. 본인 계정 바이너리 레그 확인
- 바이너리 트리 시각화: 좌측 레그 / 우측 레그 (React Flow)
- 각 레그별 멤버 수, 누적 볼륨 수치 표시
- 노드 클릭 시 해당 멤버 요약 정보 팝업
- 레그 불균형 경고 표시 (약한 쪽 레그 강조)

### 3-6. 레그 화면 *(메인에서 분기)*
- 전체 다운라인 목록 (좌/우 구분)
- 각 멤버의 가입일, 직급, 이번 달 활동 여부
- 검색 및 필터 기능 (직급별, 활동 상태별)

### 3-7. 계정 직급 확인 페이지 *(메인에서 분기)*
- 직급 체계 시각화
- 현재 직급 + 다음 직급까지 조건 달성률
- 직급별 profit sharing 비율 명시
- 이력: 직급 변동 날짜 기록
